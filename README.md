# B-Körkortsappen 🏎️

A Swedish B-driver's license theory training app. Study questions by category, take timed practice quizzes, track your progress with stats and gamification, and browse road signs — all in Swedish.

## Features

- **Study mode** — Practice questions per category with instant feedback
- **Quiz mode** — Timed mock exams (65 questions, 50 min, pass at 52)
- **Statistics** — Track accuracy, streaks, XP, levels, and badges
- **Road signs** — Browse and learn Swedish traffic signs
- **Bookmarks** — Save tricky questions for later review
- **PWA** — Installable, works offline

## Getting Started

```sh
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command           | Description                |
| ----------------- | -------------------------- |
| `npm run dev`     | Start dev server           |
| `npm run build`   | Production build           |
| `npm run preview` | Preview production build   |
| `npm run lint`    | Lint with ESLint           |
| `npm run test`    | Run tests                  |
